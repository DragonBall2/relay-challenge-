챌린지 진행 방법

1. 코딩 에이전트 설치

위 가이드를 참고하여 Open Code 또는 Roo Code를 설치합니다.

2. challenge_data.dat 다운로드

여기에서 데이터 파일을 다운로드하여 작업 폴더에 저장합니다.

3. 사이트 로그인

이전 주자에게 받은 Knox ID와 비밀번호로 로그인합니다.

4. 문제 풀기

표시된 문제를 코딩 에이전트에 그대로 입력하면, 에이전트가 challenge_data.dat를 분석하는 코드를 작성하고 실행하여 답을 알려줍니다.

5. 답안 제출

에이전트가 알려준 답을 사이트에 제출합니다. 틀리면 횟수 제한 없이 재시도 가능합니다.

6. 바톤 터치!

정답 제출 후 표시되는 다음 주자 Knox ID와 비밀번호를 전달하고, 설치를 도와주세요.
