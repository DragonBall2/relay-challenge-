Roo Code 추천

VS Code 확장 프로그램으로 설치하는 코딩 에이전트

설치 방법
  1. VS Code를 실행합니다
  2. Extensions (Ctrl+Shift+X) 을 엽니다
  3. Roo Code를 검색하고 Install을 클릭합니다
  4. 사이드바에서 Roo Code 아이콘을 클릭하여 활성화합니다
  5. API Provider 설정에서 Codemate에서 발급한 Access Token을 입력합니다
참고 링크
  • Roo Code 설치 문서
  • Codemate (Access Token 발급)

Open Code

터미널 기반 코딩 에이전트

설치 방법
  1. 터미널을 엽니다
  2. 다음 명령어로 설치합니다:
    npm install -g @anthropic-ai/opencode
  3. 또는 직접 실행:
    npx @anthropic-ai/opencode
  4. Codemate에서 발급한 Access Token으로 API 키를 설정합니다
참고 링크
  • Open Code 설치 문서
  • Codemate (Access Token 발급)

챌린지 진행 방법

1. 코딩 에이전트 설치

위 가이드를 참고하여 Roo Code 또는 Open Code를 설치합니다.

2. challenge_data.txt 다운로드

여기에서 데이터 파일을 다운로드하여 작업 폴더에 저장합니다.

3. 사이트 로그인

이전 주자에게 받은 비밀번호와 본인의 Knox-ID로 로그인합니다.

4. 문제 풀기

표시된 문제를 코딩 에이전트에 그대로 입력하면, 에이전트가 challenge_data.txt를 분석하는 코드를 작성하고 실행하여 답을 알려줍니다.

5. 답안 제출

에이전트가 알려준 답을 사이트에 제출합니다. 틀리면 횟수 제한 없이 재시도 가능합니다.

6. 바톤 터치!

정답 제출 후 표시되는 다음 주자 정보와 비밀번호를 전달하고, 설치를 도와주세요.
