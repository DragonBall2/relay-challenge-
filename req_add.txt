# 릴레이 챌린지 추가 요구사항 (이번 세션)

## 1. 포트 변경
- 기본 포트 5000 → 8080으로 변경
- app.py 및 setup_server.sh 반영

## 2. 조별 첫번째 주자 고정 기능
- CSV 파일에 group 컬럼 추가 (1~10: 해당 조의 첫번째 주자로 고정)
- group 컬럼이 없거나 비어있으면 기존처럼 랜덤 배정
- init_db.py 수정

## 3. 디자인 변경 (다크 → 라이트 테마)
- 전체 배경: 어두운 남색 → 밝은 연청색
- 카드: 흰색 배경
- 텍스트: 진한 남색 (가독성 향상)
- 네비게이션 바: 삼성 블루 유지

## 4. 설치 가이드 페이지 변경
- Open Code → 추천으로 변경 (상단 배치)
- Roo Code → 추천 제거 (하단 배치)
- 두 카드 모두 설치 방법 제거, 참고 링크만 유지

## 5. 경품 안내 추가
- 메인 페이지 및 리더보드에 경품 표시
- 완주 던킨쿠폰 항목 추가
- 완주 카드 배경색: 밝은 하늘색으로 변경

## 6. 경품 금액 변경
- 1등: 잠바주스 9만원 → 12만원
- 2등: 잠바주스 6만원 → 9만원
- 3등: 잠바주스 3만원 → 6만원
- 완주: 던킨쿠폰 2만원 → 4만원

## 7. 성공 페이지 개선
- 다음 주자 Knox ID 표시 추가 (이름 + Knox ID + 비밀번호 모두 표시)
- 안내 문구에서 PIN 관련 내용 제거

## 8. 부재 주자 뒤로 미루기 기능
- 관리자 대시보드에 "뒤로 미루기" 버튼 추가
- 해당 주자를 조의 맨 마지막 순서로 이동
- 뒤에 있던 주자들 순서 1씩 당겨짐
- active 상태였으면 다음 주자 자동 활성화
- 뒤로 미루기 시 started_at 및 attempts 초기화

## 9. 관리자 대시보드 시각 표시
- 시작 시각, 완료 시각 컬럼 추가
- KST(+9시간) 변환하여 표시
